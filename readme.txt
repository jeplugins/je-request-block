Collect and manage feature requests with a beautiful voting system.

== Description ==

**JE Request Block** is a Gutenberg block that helps you collect feature requests from your users with a built-in voting system.

Perfect for:

* **SaaS Products** - Gather user feedback and feature ideas
* **Startups** - Understand what users want most
* **Software Companies** - Prioritize development based on votes
* **Plugin/Theme Developers** - Let users suggest features
* **Community Sites** - Crowdsource ideas from members

### Features

* **Submit Form** - Users can submit feature requests easily
* **Voting System** - Upvote to prioritize features
* **Status Badges** - Pending, Planned, In Progress, Completed, Rejected
* **Filter & Sort** - Filter by status, sort by votes or date
* **2 Layouts** - List and Cards view
* **Fully Responsive** - Looks great on all devices
* **Lightweight** - No bloat, fast loading

== Installation ==

1. Install the plugin via WordPress plugins screen or upload to `/wp-content/plugins/je-request-block`.
2. Activate "JE Request Block" from the Plugins menu.
3. Edit any page or post, click the + button and search for "Feature Request".
4. Insert the block and start collecting feedback from your users!
